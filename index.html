Atlas Task Orchestrator

How to Use

1. Create: Click "Add New Task" to generate a simulated asynchronous job.
2. Observe: Only 3 tasks run at once (Concurrency Limit). Others wait in the queue.
3. Control: Use the Pause, Resume, or Cancel buttons on each card to manage tasks in real-time.
4. Persist: Refresh the page. Completed tasks are saved automatically to your local storage.
